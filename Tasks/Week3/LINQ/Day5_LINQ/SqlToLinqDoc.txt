# SQL-to-LINQ Mapping Document

## Overview

This document demonstrates how common SQL queries can be converted into LINQ queries in C#.

---

# 1. WHERE Clause

## SQL

```sql
SELECT *
FROM Employees
WHERE Department = 'IT';
```

## LINQ

```csharp
employees.Where(e => e.Department == "IT");
```

---

# 2. SELECT (Projection)

## SQL

```sql
SELECT Name
FROM Employees;
```

## LINQ

```csharp
employees.Select(e => e.Name);
```

---

## SQL

```sql

FROM Employees

```

# 4. ORDER BY Descending

## SQL

```sql

# 5. GROUP BY

## SQL

```sql
SELECT Department,
       COUNT(*)
FROM Employees
GROUP BY Departme*t;
```

## LINQ

```*sharp
employees
    .GroupBy(e => *.Department)
    .Select(g => new
*   {
        Department = g.Key,
 *      Count = g.Count()
    });
``*

# 6. INNER JOIN

## SQL

``*sql
SELECT e.Name,
       d.Depart*entName
FROM Employees e
INNER JOI* Departments d
ON e.DepartmentId =*d.DepartmentId;
```

## LINQ

```c*harp
from e in employees
join d in*departments
on e.DepartmentId equa*s d.DepartmentId
select new
{
    *.Name,
    d.DepartmentName
};
```*

# Conclusion

This document shows the equivalent LINQ expressions for commonly used SQL operations. LINQ helps developers write database-like queries directly in C# using strongly typed collections and objects.

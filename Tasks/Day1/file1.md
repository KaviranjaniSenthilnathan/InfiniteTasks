# C# Learning – Day 1: Setup & C# Basics

## Overview

This project is part of my .NET learning journey and focuses on building a strong foundation in C# programming. It contains multiple console applications that demonstrate C# fundamentals, including variables, data types, operators, control flow statements, loops, methods, and basic problem-solving techniques.

## Setup Instructions

### Prerequisites

- Visual Studio 2022
- .NET SDK (Latest LTS)

### Steps

1. Clone the repository.

2. Open the solution in Visual Studio.

3. Build the project.

4. Press **Ctrl + F5** or click **Start**.

5. Select and run the required program.

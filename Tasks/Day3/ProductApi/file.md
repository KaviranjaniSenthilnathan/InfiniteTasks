# Product API

## Overview

Product API is a simple ASP.NET Core Web API developed to understand REST API fundamentals. The project performs CRUD operations using in-memory data and demonstrates routing, controllers, and HTTP methods.

---

## Technologies Used

- ASP.NET Core Web API
- C#
- Swagger / OpenAPI
- In-Memory Collection

---

## Features

- Get All Products
- Get Product By ID
- Add Product
- Update Product
- Delete Product

---

## Project Structure

- Controllers
- Models
- Program.cs

---

## Setup Instructions

### Prerequisites

- Visual Studio 2022
- .NET SDK (Latest LTS)

### Steps

1. Clone the repository.

2. Open the solution in Visual Studio.

3. Build the project.

4. Press F5 to run.

5. Swagger will open automatically.

---

## API Endpoints

### Get All Products

GET /api/Product

---

### Get Product By ID

GET /api/Product/{id}

---

### Create Product

POST /api/Product

---

### Update Product

PUT /api/Product

---

### Delete Product

DELETE /api/Product/{id}

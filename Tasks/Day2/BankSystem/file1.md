# Bank Account System

## Overview

Bank Account System is a C# Console Application developed to demonstrate the core concepts of Object-Oriented Programming (OOP). The application simulates basic banking operations such as depositing money, withdrawing money, and checking account balance using classes, inheritance, constructors, and encapsulation.

---

## Technologies Used

- C#
- .NET SDK (Latest LTS)
- Visual Studio 2022
- Console Application

---

## Features

- Create Bank Account

- Check Account Balance

## Setup Instructions

### Prerequisites

- Visual Studio 2022
- .NET SDK (Latest LTS)

### Steps

1. Clone the repository.

2. Open the solution in Visual Studio.

3. Build the project.

4. Press **Ctrl + F5** or click **Start**.

5. Follow the console menu to perform banking operations.

## Functionalities

### Create Account

Creates a new bank account.

---

### Deposit Money

Deposits a specified amount into the account.

---

### Withdraw Money

Withdraws money from the account after checking the available balance.

---

### Check Balance

Displays the current account balance.

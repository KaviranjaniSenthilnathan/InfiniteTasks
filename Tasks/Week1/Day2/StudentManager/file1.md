# Student List Manager

## Overview

Student List Manager is a C# Console Application developed to practice Collections and LINQ. The application allows users to manage student records by adding students, displaying student information, filtering students based on marks, and sorting students by name.

---

## Technologies Used

- C#
- .NET SDK (Latest LTS)
- Visual Studio 2022
- Console Application
- LINQ

---

## Features

- Add Student
- Display All Students
- Filter Students by Marks
- Sort Students by Name
- Store Student Records using List<T>
- Query Data using LINQ

---

- Program.cs

The project contains:

- Student class

- Display Students functionality
- Filter Students by Marks
- Sort Students by Name

## Setup Instructions

### Prerequisites

- Visual Studio 2022
- .NET SDK (Latest LTS)

### Steps

1. Clone the repository.

2. Open the solution in Visual Studio.

3. Build the project.

4. Press **Ctrl + F5** or click **Start**.

5. Follow the console menu to manage student records.

---

## Functionalities

### Add Student

Adds a new student to the student list.

---

### Display All Students

Displays all student records.

---

### Filter Students by Marks

Displays students whose marks satisfy the specified condition using LINQ.

---

### Sort Students by Name

Sorts and displays students alphabetically using LINQ.
